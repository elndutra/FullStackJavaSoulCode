Collapse
Como funciona O plugin JavaScript de colapso é usado para mostrar e ocultar conteúdo. Botões ou âncoras são usados ​​como gatilhos mapeados para elementos específicos que você alterna. Recolher um elemento animará a altura de seu valor atual para 0. Dado como o CSS lida com animações, você não pode usar preenchimento em um elemento .collapse. Em vez disso, use a classe como um elemento de encapsulamento independente.
Accordion
Este é o corpo do acordeão do primeiro item. Ele é mostrado por padrão, até que o plugin de recolhimento adicione as classes apropriadas que usamos para estilizar cada elemento. Essas classes controlam a aparência geral, bem como mostrar e ocultar por meio de transições CSS. Você pode modificar tudo isso com CSS personalizado ou substituindo nossas variáveis ​​padrão. Também é importante notar que praticamente qualquer HTML pode entrar no .accordion-body, embora a transição limite o estouro.
