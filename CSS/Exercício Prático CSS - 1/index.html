O que é CSS?

CSS (Folhas de Estilo em Cascata) permite a você criar páginas web agradáveis. CSS é uma linguagem baseada em regras. — Você define regras especificando grupos de estilo que devem ser aplicados para elementos particulares ou grupos de elementos na sua página web. Por exemplo, pra essa ativade será alterado o título com a cor darkcyan.
